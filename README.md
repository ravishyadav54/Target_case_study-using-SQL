# Target_case_study-using-SQL

📌 Overview

This project focuses on analyzing the e-commerce operations of Target in Brazil using SQL. The dataset contains over 100,000 orders (2016–2018) and provides insights into customer behavior, sales trends, logistics performance, and payment patterns.
The goal of this project is to extract actionable business insights and provide data-driven recommendations to improve operational efficiency, customer satisfaction, and revenue.

📂 Dataset Description

The analysis is based on 8 CSV files:

customers.csv,
sellers.csv,
order_items.csv,
geolocation.csv,
payments.csv,
reviews.csv,
orders.csv,
products.csv,

These datasets include information about:

Customers and locations,
Orders and delivery timelines,
Payments and pricing,
Products and sellers,
Reviews and ratings

🎯 Problem Statement

As a data analyst at Target, the objective is to:

Perform exploratory data analysis (EDA),
Identify trends and patterns in orders,
Analyze logistics and delivery performance,
Evaluate financial metrics like order value and freight,
Understand customer and payment behavior,
Provide actionable business recommendations

🛠️ Tools & Technologies
SQL (BigQuery),
Data Cleaning & Transformation,
Aggregations, Joins, CTEs,
Date & Time Functions
